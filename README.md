# Project for Client

## Cara Jalanin

### Menjalankan server terlebih dahulu 

Buka terminal dan pergi ke folder backend dengan perintah berikut:
```bash
cd backend
```

Lalu jalanin perintah berikut:
```bash
npm run dev
```

### Menjalankan frontend / tampilan awal

Buka terminal / tambah terminal di code editor mu lalu pergi ke folder frontend dengan perintah berikut:
```bash
cd frontend
```

Lalu jalanin perintah berikut:
```bash
npm run dev
```
